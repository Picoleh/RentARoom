# RentRoom Project

This repository currently contains a React + TypeScript frontend app (Vite).

## Project structure

- `Frontend/` - React application
- `Backend/` - backend placeholder

## Prerequisites

Install these tools before running the project:

- Node.js 20+ (recommended)
- npm (comes with Node.js)

## Run the frontend locally

From the workspace root:

```powershell
cd Frontend
npm install
npm run dev
```

After starting, open the local URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```powershell
cd Frontend
npm run build
```

The production files are generated in `Frontend/dist`.

## Preview production build locally

```powershell
cd Frontend
npm run preview
```

## Lint the frontend

```powershell
cd Frontend
npm run lint
```

## Notes

- The `Backend/` directory is currently empty, so there is no backend run command yet.
- The existing `Frontend/README.md` is the default Vite template; this root README is the main run guide for this repository.
